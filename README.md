# FICTIONAL RESTAURANT WEBSITE


# AUTHORS
Victor Gichobi

# DESCRIPTION
A modern, responsive restaurant website built using HTML, CSS, Bootstrap, and Font Awesome. The website showcases a fictional restaurant with an attractive hero section, menu display, about section, contact information, and social media integration.

## Features
Responsive design for desktop, tablet, and mobile devices
Attractive hero section with high-quality restaurant imagery
Interactive navigation bar

## Technologies used
-Html
-Css
-Bootstrap 5

# Usage
-Download the project
-Open folder
-Run index.html

## Future Enhancements
-Online reservation system
-Food ordering functionality
-Customer reviews section
-Gallery page
